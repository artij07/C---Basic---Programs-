
# C Basic Programs

This repository contains simple **C programming exercises** for beginners.  
Each file demonstrates a basic concept of programming such as loops, conditionals, functions, etc.

## 📂 Programs Included
1. `hello_world.c` - Print "Hello World"
2. `sum_two_numbers.c` - Add two numbers
3. `even_or_odd.c` - Check even or odd
4. `largest_of_two.c` - Find the largest of two numbers
5. `calculator.c` - Simple calculator using switch
6. `factorial.c` - Factorial of a number
7. `fibonacci.c` - Fibonacci series
8. `reverse_number.c` - Reverse a number
9. `palindrome.c` - Check palindrome number
10. `prime_numbers.c` - Print prime numbers up to N

## 🛠️ How to Run
Compile the code using `gcc` and run:

```bash
gcc program_name.c -o program_name
./program_name
